# 📘 **68\_normas\_seguridad\_pinzas\_ER.md**

**Normas oficiales de seguridad para pinzas ER — CNC‑CAT / LINCAT**

## ⭐ 1. Propósito

Definir las **normas de seguridad para el uso de pinzas ER** en máquinas CNC‑CAT, garantizando:

- seguridad del operador

- integridad del portaherramientas

- integridad de la herramienta

- estabilidad del mecanizado

- precisión geométrica

- cumplimiento industrial

Las pinzas ER son el sistema de sujeción más común en CNC. Si fallan, la herramienta sale despedida. Estas normas son **críticas** y deben cumplirse SIEMPRE.

## ⭐ 2. Principios fundamentales

- **La pinza ER es un elemento de precisión** — cualquier deformación arruina el mecanizado.

- **La limpieza es obligatoria** — suciedad = desalineación.

- **El torque es ley** — demasiado poco = herramienta floja; demasiado = deformación.

- **La compatibilidad es crítica** — diámetro incorrecto = accidente.

## ⭐ 3. Componentes cubiertos

- **Pinza ER**

- **Tuerca ER**

- **Portapinzas ER**

- **Herramienta**

## ⭐ 4. Requisitos previos

Antes de montar una pinza ER:

- **Revisar limpieza**

- **Revisar desgaste**

- **Revisar deformación**

- **Revisar diámetro de herramienta**

- **Revisar torque de tuerca**

## ⭐ 5. Seguridad en montaje

- **Prohibido montar pinzas sucias**

- **Prohibido montar pinzas deformadas**

- **Prohibido montar herramientas fuera de tolerancia**

- **Prohibido montar sin torque adecuado**

- Validación obligatoria:

  - concentricidad

  - fijación

  - vibración

## ⭐ 6. Seguridad en operación

- **Prohibido usar RPM excesivas**

- **Prohibido usar herramientas desbalanceadas**

- **Prohibido usar herramientas demasiado largas**

- **Prohibido usar pinzas con desgaste térmico**

- Validación obligatoria:

  - ruido

  - vibración

  - temperatura

## ⭐ 7. Seguridad en tuercas ER

- **Prohibido usar tuercas sin lubricación mínima**

- **Prohibido usar tuercas con rosca dañada**

- **Prohibido usar tuercas sin torque adecuado**

- Validación obligatoria:

  - suavidad

  - torque

  - vibración

## ⭐ 8. Seguridad en portapinzas ER

- **Prohibido usar portapinzas sucios**

- **Prohibido usar portapinzas con óxido**

- **Prohibido usar portapinzas deformados**

- Validación obligatoria:

  - contacto

  - concentricidad

  - vibración

## ⭐ 9. Prohibiciones absolutas

- usar pinzas sucias

- usar pinzas deformadas

- usar herramientas fuera de tolerancia

- usar tuercas sin torque

- usar portapinzas oxidados

- usar herramientas desbalanceadas

- ignorar vibración

- ignorar ruido

- ignorar temperatura

- operar sin validar

- operar sin registrar

## ⭐ 10. Objetivo final

Estas normas garantizan que las pinzas ER en CNC‑CAT sean:

- seguras

- estables

- precisas

- profesionales

- industriales

- trazables

- confiables

Son las normas oficiales de seguridad de pinzas ER del ecosistema CNC‑CAT.
