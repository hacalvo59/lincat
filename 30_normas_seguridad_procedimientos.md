# 📘 **30\_normas\_seguridad\_procedimientos.md**

**Normas oficiales de seguridad en procedimientos — CNC‑CAT / LINCAT**

# ⭐ 1. Propósito

Definir las **normas de seguridad aplicables a todos los procedimientos operativos**, técnicos y administrativos del ecosistema CNC‑CAT, garantizando:

- uniformidad

- trazabilidad

- seguridad

- cumplimiento industrial

- reducción de errores humanos

- operación profesional

Estas normas son **obligatorias** para cualquier procedimiento documentado o ejecutado.

# ⭐ 2. Principios fundamentales

### ✔️ 2.1 Todo procedimiento debe ser seguro

Si un procedimiento no es seguro → no se ejecuta.

### ✔️ 2.2 Todo procedimiento debe ser claro

Nada de ambigüedades, interpretaciones o improvisaciones.

### ✔️ 2.3 Todo procedimiento debe ser reproducible

Dos técnicos distintos deben obtener el mismo resultado.

### ✔️ 2.4 Todo procedimiento debe ser trazable

Debe quedar registrado quién lo ejecutó, cuándo y cómo.

# ⭐ 3. Estructura obligatoria de un procedimiento

Todo procedimiento CNC‑CAT debe incluir:

### ✔️ 3.1 Identificación

- nombre

- código

- versión

- autor

- fecha

### ✔️ 3.2 Objetivo

Qué se busca lograr.

### ✔️ 3.3 Alcance

A qué máquinas, módulos o usuarios aplica.

### ✔️ 3.4 Requisitos previos

- formación

- permisos

- herramientas

- EPP

- estado de la máquina

### ✔️ 3.5 Pasos detallados

Enumerados, claros, sin ambigüedad.

### ✔️ 3.6 Criterios de éxito

Cómo saber que el procedimiento se ejecutó correctamente.

### ✔️ 3.7 Acciones en caso de fallo

Qué hacer si algo sale mal.

### ✔️ 3.8 Registro

Qué debe documentarse al finalizar.

# ⭐ 4. Seguridad en procedimientos operativos

### ✔️ 4.1 Prohibido ejecutar procedimientos sin EPP

Siempre:

- gafas

- calzado

- protección auditiva

### ✔️ 4.2 Prohibido ejecutar procedimientos con la máquina activa

Excepto cuando el procedimiento lo requiera explícitamente.

### ✔️ 4.3 Prohibido saltarse pasos

Los pasos son obligatorios.

### ✔️ 4.4 Prohibido improvisar

Si un paso no aplica → detener y consultar.

# ⭐ 5. Seguridad en procedimientos técnicos

### ✔️ 5.1 Solo personal autorizado

Técnicos o administradores.

### ✔️ 5.2 LOTO obligatorio

Bloqueo y etiquetado antes de:

- mantenimiento

- inspección

- reparación

### ✔️ 5.3 Documentación obligatoria

Registrar:

- acciones

- piezas cambiadas

- valores ajustados

- observaciones

### ✔️ 5.4 Validación posterior

Siempre:

- homing

- verificación de límites

- verificación de sensores

- verificación de servos

# ⭐ 6. Seguridad en procedimientos administrativos

### ✔️ 6.1 Control de versiones

Toda actualización debe registrarse.

### ✔️ 6.2 Control de permisos

Solo administradores pueden:

- aprobar procedimientos

- modificar procedimientos

- eliminar procedimientos

### ✔️ 6.3 Auditoría

Cada procedimiento debe tener historial completo.

# ⭐ 7. Procedimientos críticos

Los siguientes procedimientos requieren **doble validación**:

- cambio de parámetros del planner

- cambio de límites

- cambio de offsets globales

- calibración de servos

- configuración de EtherCAT

- actualización del sistema

- restauración de backups

# ⭐ 8. Procedimientos prohibidos

- ejecutar procedimientos sin documentación

- ejecutar procedimientos sin permisos

- ejecutar procedimientos sin EPP

- ejecutar procedimientos con la máquina activa (si no corresponde)

- modificar procedimientos sin aprobación

- eliminar pasos por conveniencia

- ignorar pasos de seguridad

# ⭐ 9. Procedimientos de emergencia

Todo procedimiento debe incluir:

- cómo detener la máquina

- cómo activar E‑STOP

- cómo evacuar

- cómo actuar ante fallos eléctricos

- cómo actuar ante fallos mecánicos

# ⭐ 10. Registro y trazabilidad

Cada procedimiento ejecutado debe registrar:

- operador/técnico

- fecha

- hora

- máquina

- resultado

- incidencias

- acciones correctivas

# ⭐ 11. Objetivo final

Estas normas garantizan que los procedimientos CNC‑CAT sean:

- seguros

- claros

- reproducibles

- auditables

- profesionales

- industriales

Son las normas oficiales de seguridad en procedimientos del ecosistema CNC‑CAT.
