# 📘 **45\_normas\_seguridad\_potencia.md**

**Normas oficiales de seguridad en sistemas de potencia — CNC‑CAT / LINCAT**

# ⭐ 1. Propósito

Definir las **normas de seguridad obligatorias** para instalar, operar, integrar y mantener sistemas de potencia en máquinas controladas por CNC‑CAT, garantizando:

- seguridad del operador

- integridad eléctrica

- protección del hardware

- estabilidad del sistema

- cumplimiento industrial

Los sistemas de potencia son la base energética del CNC. Un error aquí no causa un fallo… causa un accidente. Estas normas son **críticas** y deben cumplirse SIEMPRE.

# ⭐ 2. Principios fundamentales

### ✔️ **La potencia es peligrosa**

Tensiones y corrientes elevadas pueden causar daños graves.

### ✔️ **La protección es obligatoria**

Fusibles, magnetotérmicos, diferenciales, relés de seguridad.

### ✔️ **La separación es esencial**

Nunca mezclar potencia con señales de control.

### ✔️ **La documentación es ley**

Sin esquema → no se toca.

# ⭐ 3. Tipos de sistemas de potencia cubiertos

- **Alimentación principal** — 230/400V

- **Fuentes de 24V** — control y sensores

- **Drivers de servos** — alto consumo

- **Variadores/VFD** — husillo

- **Actuadores de potencia** — solenoides, bombas

- **Protecciones eléctricas**

# ⭐ 4. Requisitos previos

Antes de trabajar con potencia, el técnico debe:

- ser usuario con rol **técnico** o **administrador**

- conocer el esquema eléctrico

- verificar tensiones

- verificar corrientes

- verificar protecciones

- activar LOTO

- usar EPP completo

# ⭐ 5. Equipo de protección personal (EPP)

Obligatorio:

- guantes aislantes

- gafas de seguridad

- calzado dieléctrico

- ropa ignífuga

Opcional según operación:

- pantalla facial

- alfombra aislante

# ⭐ 6. Seguridad en alimentación principal

### ✔️ **Prohibido trabajar con tensión**

LOTO obligatorio.

### ✔️ **Prohibido usar cables no certificados**

Solo cables industriales.

### ✔️ **Prohibido mezclar fases**

Cada fase debe estar correctamente identificada.

### ✔️ Validación obligatoria

- continuidad

- aislamiento

- polaridad

- protecciones

# ⭐ 7. Seguridad en fuentes de 24V

### ✔️ **Prohibido sobrecargar la fuente**

Debe tener margen del 30%.

### ✔️ **Prohibido mezclar 24V con señales analógicas**

Evita ruido.

### ✔️ Validación obligatoria

- ripple

- caída de tensión

- estabilidad

# ⭐ 8. Seguridad en drivers de servos

### ✔️ **Prohibido conectar servos sin tierra**

Riesgo extremo.

### ✔️ **Prohibido usar drivers sin disipación**

Pueden incendiarse.

### ✔️ **Prohibido usar cables no apantallados**

Evita interferencias.

### ✔️ Validación obligatoria

- corriente

- temperatura

- vibración

# ⭐ 9. Seguridad en variadores/VFD (husillo)

### ✔️ **Prohibido operar sin filtro EMI**

Evita ruido en todo el sistema.

### ✔️ **Prohibido operar sin freno adecuado**

Evita daños mecánicos.

### ✔️ **Prohibido operar sin ventilación**

Evita sobrecalentamiento.

### ✔️ Validación obligatoria

- rampas

- corriente

- temperatura

# ⭐ 10. Seguridad en actuadores de potencia

### ✔️ **Prohibido conectar cargas inductivas sin diodo**

Evita picos de tensión.

### ✔️ **Prohibido usar relés subdimensionados**

Riesgo de incendio.

### ✔️ Validación obligatoria

- corriente

- tensión

- temperatura

# ⭐ 11. Seguridad en protecciones eléctricas

### ✔️ **Prohibido operar sin magnetotérmico**

Protección básica.

### ✔️ **Prohibido operar sin diferencial**

Protección de vida.

### ✔️ **Prohibido operar sin relé de seguridad**

E‑STOP debe ser redundante.

# ⭐ 12. Prohibiciones absolutas

- trabajar con tensión

- mezclar tensiones

- usar cables dañados

- usar protecciones incorrectas

- ignorar ruido eléctrico

- ignorar temperatura

- ignorar vibración

- ignorar olor a quemado

- operar sin validar

- operar sin registrar

# ⭐ 13. Objetivo final

Estas normas garantizan que los sistemas de potencia CNC‑CAT sean:

- seguros

- estables

- profesionales

- industriales

- trazables

- confiables

Son las normas oficiales de seguridad en potencia del ecosistema CNC‑CAT.
