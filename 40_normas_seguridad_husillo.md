# 📘 **40\_normas\_seguridad\_husillo.md**

**Normas oficiales de seguridad del husillo — CNC‑CAT / LINCAT**

# ⭐ 1. Propósito

Definir las **normas de seguridad obligatorias** para operar, configurar, mantener y diagnosticar el husillo en máquinas controladas por CNC‑CAT, garantizando:

- seguridad del operador

- integridad del husillo

- integridad de la herramienta

- estabilidad del mecanizado

- cumplimiento industrial

El husillo es uno de los elementos más peligrosos de una máquina CNC. Estas normas son **críticas** y deben cumplirse SIEMPRE.

# ⭐ 2. Principios fundamentales

### ✔️ 2.1 El husillo es un elemento de alto riesgo

Gira a miles de RPM y puede expulsar herramientas.

### ✔️ 2.2 La seguridad depende de la herramienta

Una herramienta mal fijada → accidente inmediato.

### ✔️ 2.3 La velocidad debe ser adecuada

RPM incorrectas → rotura, vibración o incendio.

### ✔️ 2.4 Solo personal cualificado puede operar el husillo

No es un componente trivial.

# ⭐ 3. Requisitos previos

Antes de operar el husillo, el técnico debe:

- ser usuario con rol **operador**, **técnico** o **administrador**

- revisar estado de la herramienta

- revisar fijación

- revisar cono y pinza

- revisar lubricación (si aplica)

- revisar sensores

- revisar ventilación

- revisar parámetros de velocidad

# ⭐ 4. Equipo de protección personal (EPP)

Obligatorio:

- gafas de seguridad

- calzado de seguridad

- ropa ajustada

- pelo recogido

Opcional según operación:

- protección auditiva

- pantalla facial

# ⭐ 5. Seguridad en herramientas

### ✔️ 5.1 Prohibido usar herramientas dañadas

Incluye:

- filos rotos

- desgaste excesivo

- vibración visible

### ✔️ 5.2 Prohibido usar herramientas sin fijación correcta

La pinza debe estar:

- limpia

- sin virutas

- sin aceite

- bien apretada

### ✔️ 5.3 Prohibido usar herramientas desbalanceadas

A altas RPM → riesgo extremo.

### ✔️ 5.4 Validación obligatoria

Después de montar herramienta:

- longitud

- diámetro

- fijación

- limpieza

# ⭐ 6. Seguridad en operación del husillo

### ✔️ 6.1 Prohibido encender el husillo con la puerta abierta

Si la máquina tiene cerramiento.

### ✔️ 6.2 Prohibido acercar manos al husillo

Incluso a baja velocidad.

### ✔️ 6.3 Prohibido cambiar herramienta con husillo activo

Debe estar completamente detenido.

### ✔️ 6.4 Prohibido operar sin ventilación

Evita sobrecalentamiento.

### ✔️ 6.5 Prohibido operar sin sensor de velocidad

Debe reportar RPM reales.

# ⭐ 7. Seguridad en velocidades y avances

### ✔️ 7.1 Prohibido usar RPM incorrectas

Cada herramienta tiene su rango.

### ✔️ 7.2 Prohibido usar avances excesivos

Evita rotura de herramienta.

### ✔️ 7.3 Prohibido usar rampas agresivas

Aceleraciones bruscas → vibración.

### ✔️ 7.4 Validación obligatoria

- vibración

- ruido

- temperatura

- estabilidad

# ⭐ 8. Seguridad en frenado

### ✔️ 8.1 Prohibido frenar el husillo manualmente

Nunca tocar la herramienta.

### ✔️ 8.2 Prohibido usar frenado mecánico no certificado

Solo freno del driver o del VFD.

### ✔️ 8.3 Validación obligatoria

- tiempo de frenado

- temperatura

- vibración

# ⭐ 9. Seguridad en mantenimiento del husillo

### ✔️ 9.1 Prohibido realizar mantenimiento con tensión

Debe estar apagado y bloqueado (LOTO).

### ✔️ 9.2 Prohibido desmontar el husillo sin documentación

Cada modelo es distinto.

### ✔️ 9.3 Prohibido lubricar con productos no certificados

Evita contaminación.

### ✔️ 9.4 Validación obligatoria

Después de mantenimiento:

- ruido

- vibración

- temperatura

- precisión

# ⭐ 10. Seguridad en sensores del husillo

### ✔️ 10.1 Sensor de velocidad

Debe funcionar correctamente.

### ✔️ 10.2 Sensor de temperatura

Evita sobrecalentamiento.

### ✔️ 10.3 Sensor de carga (si aplica)

Evita rotura de herramienta.

### ✔️ 10.4 Validación obligatoria

- lectura estable

- respuesta rápida

- alarmas correctas

# ⭐ 11. Prohibiciones absolutas

- operar sin EPP

- operar con herramienta dañada

- operar con herramienta floja

- operar sin sensor de velocidad

- operar sin ventilación

- operar con puerta abierta

- operar sin validar

- operar sin registrar

- frenar manualmente

- usar herramientas desbalanceadas

- usar RPM incorrectas

- ignorar vibración

- ignorar ruido

- ignorar temperatura

# ⭐ 12. Objetivo final

Estas normas garantizan que el husillo CNC‑CAT sea operado de forma:

- segura

- estable

- profesional

- industrial

- confiable

Son las normas oficiales de seguridad del husillo del ecosistema CNC‑CAT.
