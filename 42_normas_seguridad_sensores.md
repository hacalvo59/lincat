# 📘 **42\_normas\_seguridad\_sensores.md**

**Normas oficiales de seguridad para sensores — CNC‑CAT / LINCAT**

# ⭐ 1. Propósito

Definir las **normas de seguridad obligatorias** para instalar, calibrar, integrar, operar y mantener sensores en máquinas controladas por CNC‑CAT, garantizando:

- seguridad del operador

- integridad del hardware

- precisión de lectura

- estabilidad del sistema

- cumplimiento industrial

Los sensores son la “percepción” del CNC. Si fallan, la máquina queda ciega. Estas normas son **críticas** y deben cumplirse SIEMPRE.

# ⭐ 2. Principios fundamentales

### ✔️ 2.1 Un sensor defectuoso es un riesgo inmediato

Puede causar colisiones, errores de homing o fallos de seguridad.

### ✔️ 2.2 La lectura debe ser confiable

Sin ruido, sin interferencias, sin falsos positivos.

### ✔️ 2.3 La instalación debe ser correcta

Altura, orientación, distancia, fijación.

### ✔️ 2.4 Solo personal cualificado puede manipular sensores

No es un componente trivial.

# ⭐ 3. Tipos de sensores cubiertos

### ✔️ 3.1 Sensores de posición

- finales de carrera

- homing switches

- sensores inductivos

- sensores ópticos

### ✔️ 3.2 Sensores de seguridad

- puertas

- barreras

- E‑STOP externos

### ✔️ 3.3 Sensores analógicos

- temperatura

- corriente

- vibración

### ✔️ 3.4 Sensores de herramienta

- longitud

- diámetro

- rotura

### ✔️ 3.5 Sensores de entorno

- flujo de refrigerante

- presión

- vacío

# ⭐ 4. Requisitos previos

Antes de trabajar con sensores, el técnico debe:

- ser usuario con rol **técnico** o **administrador**

- revisar documentación del sensor

- revisar cableado

- revisar conectores

- revisar alimentación

- revisar blindaje

- revisar parámetros de lectura

# ⭐ 5. Equipo de protección personal (EPP)

Obligatorio:

- gafas de seguridad

- calzado de seguridad

- ropa ajustada

Opcional según operación:

- guantes

- pantalla facial

# ⭐ 6. Seguridad en instalación de sensores

### ✔️ 6.1 Prohibido instalar sensores con la máquina energizada

Debe estar apagada y bloqueada (LOTO).

### ✔️ 6.2 Prohibido instalar sensores sin fijación correcta

Evita vibraciones y lecturas falsas.

### ✔️ 6.3 Prohibido instalar sensores sin distancia adecuada

Cada sensor tiene su rango.

### ✔️ 6.4 Prohibido instalar sensores sin blindaje

Evita interferencias electromagnéticas.

### ✔️ 6.5 Validación obligatoria

Después de instalar:

- lectura

- repetibilidad

- histéresis

- estabilidad

# ⭐ 7. Seguridad en sensores de posición

### ✔️ 7.1 Prohibido operar con finales de carrera defectuosos

Son la última barrera de seguridad.

### ✔️ 7.2 Prohibido operar con homing switches inconsistentes

El homing debe ser preciso.

### ✔️ 7.3 Prohibido usar sensores inductivos sin distancia correcta

Evita falsos positivos.

### ✔️ 7.4 Validación obligatoria

- repetibilidad

- histéresis

- distancia de activación

# ⭐ 8. Seguridad en sensores de seguridad

### ✔️ 8.1 Prohibido operar con puertas sin sensor

Debe detectarse apertura.

### ✔️ 8.2 Prohibido operar con barreras defectuosas

Evita atrapamientos.

### ✔️ 8.3 Prohibido operar sin E‑STOP externo funcional

Debe detener la máquina inmediatamente.

### ✔️ 8.4 Validación obligatoria

- respuesta

- redundancia

- alarmas

# ⭐ 9. Seguridad en sensores analógicos

### ✔️ 9.1 Prohibido usar sensores sin calibración

Especialmente temperatura y corriente.

### ✔️ 9.2 Prohibido usar sensores sin filtrado

Evita ruido eléctrico.

### ✔️ 9.3 Prohibido usar sensores sin referencia

Debe existir un valor base.

### ✔️ 9.4 Validación obligatoria

- estabilidad

- precisión

- desviación

# ⭐ 10. Seguridad en sensores de herramienta

### ✔️ 10.1 Prohibido medir herramientas dañadas

Puede romper el sensor.

### ✔️ 10.2 Prohibido medir herramientas sin limpiar

Virutas → medición incorrecta.

### ✔️ 10.3 Prohibido usar sensor sin fijación correcta

Evita colisiones.

### ✔️ 10.4 Validación obligatoria

- longitud

- diámetro

- repetibilidad

# ⭐ 11. Seguridad en sensores de entorno

### ✔️ 11.1 Prohibido operar sin flujo de refrigerante

Evita sobrecalentamiento.

### ✔️ 11.2 Prohibido operar sin presión adecuada

Evita fallos de sujeción.

### ✔️ 11.3 Prohibido operar sin vacío correcto

Evita desprendimiento de piezas.

# ⭐ 12. Prohibiciones absolutas

- usar sensores dañados

- usar sensores sucios

- usar sensores sin calibración

- usar sensores sin fijación correcta

- usar sensores sin blindaje

- ignorar lecturas inestables

- ignorar ruido eléctrico

- ignorar alarmas

- operar sin validar

- operar sin registrar

# ⭐ 13. Objetivo final

Estas normas garantizan que los sensores CNC‑CAT sean:

- seguros

- precisos

- profesionales

- industriales

- trazables

- confiables

Son las normas oficiales de seguridad para sensores del ecosistema CNC‑CAT.
