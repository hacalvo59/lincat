# 📘 **26\_normas\_seguridad\_entorno.md**

**Normas oficiales de seguridad del entorno — CNC‑CAT / LINCAT**

# ⭐ 1. Propósito

Definir las **normas de seguridad del entorno físico, ambiental y organizativo** donde opera una máquina controlada por CNC‑CAT, garantizando:

- seguridad del operador

- seguridad de la máquina

- seguridad del entorno

- cumplimiento industrial

- prevención de riesgos ambientales

Estas normas son **obligatorias** en cualquier instalación profesional.

# ⭐ 2. Principios fundamentales

### ✔️ 2.1 El entorno es parte del sistema

Una máquina segura en un entorno inseguro sigue siendo peligrosa.

### ✔️ 2.2 El entorno debe ser controlado

Nada de improvisaciones.

### ✔️ 2.3 El entorno debe minimizar riesgos

Iluminación, ventilación, orden, señalización.

### ✔️ 2.4 El entorno debe permitir evacuación rápida

Rutas despejadas y accesibles.

# ⭐ 3. Requisitos del espacio físico

### ✔️ 3.1 Suelo seguro

Debe ser:

- plano

- limpio

- antideslizante

- libre de obstáculos

### ✔️ 3.2 Espacio libre alrededor de la máquina

Mínimo recomendado:

Código

```
`1 metro en todos los lados`
```

### ✔️ 3.3 Iluminación adecuada

Evitar sombras en:

- mesa

- herramienta

- pieza

- panel de control

### ✔️ 3.4 Ventilación correcta

Evitar acumulación de:

- polvo

- humo

- vapores

- calor

# ⭐ 4. Control ambiental

### ✔️ 4.1 Temperatura estable

Rango recomendado:

Código

```
`15°C – 30°C`
```

### ✔️ 4.2 Humedad controlada

Evitar condensación.

### ✔️ 4.3 Evitar polvo excesivo

Especialmente en:

- armario eléctrico

- ventiladores

- guías lineales

### ✔️ 4.4 Evitar vibraciones externas

La máquina debe estar sobre superficie estable.

# ⭐ 5. Señalización obligatoria

### ✔️ 5.1 Señal de E‑STOP

Visible desde cualquier ángulo.

### ✔️ 5.2 Señal de riesgo mecánico

Zonas con partes móviles.

### ✔️ 5.3 Señal de riesgo eléctrico

Armario eléctrico y cables de potencia.

### ✔️ 5.4 Señal de uso obligatorio de EPP

Gafas, protección auditiva, etc.

### ✔️ 5.5 Señal de evacuación

Rutas claramente marcadas.

# ⭐ 6. Gestión del entorno

### ✔️ 6.1 Orden y limpieza

Aplicar metodología **5S**:

- Clasificar

- Ordenar

- Limpiar

- Estandarizar

- Sostener

### ✔️ 6.2 Herramientas organizadas

Nada de herramientas sueltas en la mesa.

### ✔️ 6.3 Cables organizados

Uso obligatorio de:

- canaletas

- bridas

- pasacables

### ✔️ 6.4 Contenedores de virutas

Nunca permitir acumulación excesiva.

# ⭐ 7. Seguridad contra incendios

### ✔️ 7.1 Extintor obligatorio

Tipo recomendado:

Código

```
`CO₂ o polvo ABC`
```

### ✔️ 7.2 Extintor accesible

A menos de 5 metros.

### ✔️ 7.3 Prohibido bloquear extintores

Ni con cajas, ni con herramientas.

### ✔️ 7.4 Prohibido almacenar líquidos inflamables cerca

Especialmente:

- disolventes

- aceites

- aerosoles

# ⭐ 8. Seguridad eléctrica del entorno

### ✔️ 8.1 Tomas de corriente certificadas

Nada de regletas baratas.

### ✔️ 8.2 Toma de tierra obligatoria

Toda la instalación debe estar correctamente conectada.

### ✔️ 8.3 Prohibido cables sueltos

Riesgo de tropiezos y cortocircuitos.

### ✔️ 8.4 Armario eléctrico cerrado

Siempre.

# ⭐ 9. Seguridad organizativa

### ✔️ 9.1 Solo personal autorizado

Nada de visitas sin supervisión.

### ✔️ 9.2 Registro de acceso

Control de quién entra y sale.

### ✔️ 9.3 Formación obligatoria

Todo operador debe estar formado en:

- seguridad

- operación

- emergencias

### ✔️ 9.4 Simulacros periódicos

Evacuación y respuesta a emergencias.

# ⭐ 10. Situaciones de emergencia

### ✔️ 10.1 Rutas de evacuación despejadas

Siempre.

### ✔️ 10.2 E‑STOP accesible

Nunca bloquearlo.

### ✔️ 10.3 Extintor accesible

Nunca bloquearlo.

### ✔️ 10.4 Prohibido correr

Evitar accidentes adicionales.

# ⭐ 11. Prohibiciones absolutas

- operar en un entorno sucio

- operar sin iluminación

- operar sin ventilación

- operar con cables sueltos

- operar con rutas de evacuación bloqueadas

- almacenar líquidos inflamables cerca

- permitir acceso a personas no autorizadas

- ignorar señalización

- operar en entornos con vibraciones externas

# ⭐ 12. Objetivo final

Estas normas garantizan que el entorno donde opera CNC‑CAT sea:

- seguro

- profesional

- industrial

- estable

- confiable

Son las normas oficiales de seguridad del entorno del ecosistema CNC‑CAT.
