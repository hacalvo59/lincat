# 📘 **13\_portada\_visual.md**

**Portada visual oficial del ecosistema CNC‑CAT / LINCAT**

# ⭐ 1. Propósito

Definir la **portada visual oficial** que se utiliza en:

- documentación PDF

- manuales técnicos

- presentaciones

- informes

- material corporativo

- fichas de producto

La portada es la **identidad visual primaria** del ecosistema CNC‑CAT.

# ⭐ 2. Elementos obligatorios de la portada

La portada debe incluir SIEMPRE:

1. **Logo CNC‑CAT**

2. **Título del documento**

3. **Subtítulo (opcional)**

4. **Versión del documento**

5. **Fecha de publicación**

6. **Autor o módulo responsable**

7. **Franja visual industrial** (color corporativo)

8. **Pie de página con copyright**

# ⭐ 3. Composición visual

### **3.1 Estructura general**

Código

```
`┌──────────────────────────────────────────────┐`

`│                 LOGO CNC‑CAT                 │`

`│                                              │`

`│                TÍTULO PRINCIPAL              │`

`│                Subtítulo opcional            │`

`│                                              │`

`│                Versión · Fecha               │`

`│                                              │`

`│──────────────────────────────────────────────│`

`│                                              │`

`│           Franja industrial azul             │`

`│                                              │`

`└──────────────────────────────────────────────┘`
```

# ⭐ 4. Colores oficiales

### **Franja industrial**

- Azul CNC‑CAT: `\#1E88E5`

### **Texto**

- Gris oscuro: `\#212121`

- Blanco (sobre franja): `\#FFFFFF`

### **Fondo**

- Blanco puro: `\#FFFFFF`

# ⭐ 5. Tipografías

### **Título**

- Roboto Bold — 36–48 px

### **Subtítulo**

- Roboto Medium — 22–28 px

### **Metadatos (versión, fecha, autor)**

- Roboto Regular — 16–18 px

### **Pie de página**

- Roboto Light — 12–14 px

# ⭐ 6. Logo CNC‑CAT

### **Reglas**

- Siempre centrado

- Nunca estirado

- Nunca con sombras

- Nunca con efectos

- Tamaño recomendado: 180–240 px de ancho

### **Variantes permitidas**

- Logo azul sobre fondo blanco

- Logo blanco sobre franja azul

# ⭐ 7. Márgenes y espaciado

- Margen superior: 80 px

- Margen inferior: 60 px

- Espaciado entre título y subtítulo: 12 px

- Espaciado entre subtítulo y metadatos: 24 px

- Espaciado entre metadatos y franja: 40 px

# ⭐ 8. Pie de página

Debe incluir:

Código

```
`© CNC‑CAT Project — Todos los derechos reservados`
```

Opcionalmente:

Código

```
`Documento interno — No distribuir sin autorización`
```

# ⭐ 9. Ejemplo de portada (texto)

Código

```
`CNC‑CAT — MOTOR CNC INDUSTRIAL`

`Guía de Arquitectura v1.0`

`Junio 2026`

`Autor: Equipo CORE`
```

# ⭐ 10. Objetivo final

La portada visual garantiza:

- identidad

- profesionalismo

- coherencia

- estética industrial

- reconocimiento inmediato

Es la portada oficial del ecosistema CNC‑CAT.
