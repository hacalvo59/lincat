# 📘 **77\_normas\_seguridad\_fresas\_desbaste.md**

**Normas oficiales de seguridad para fresas de desbaste — CNC‑CAT / LINCAT**

## ⭐ 1. Propósito

Definir las **normas de seguridad para fresas de desbaste** utilizadas en máquinas CNC‑CAT, garantizando:

- seguridad del operador

- integridad del husillo

- integridad del portaherramientas

- estabilidad del mecanizado

- máxima evacuación de viruta

- cumplimiento industrial

Las fresas de desbaste son agresivas, potentes y ruidosas. Si fallan, fallan con violencia. Estas normas son **críticas** y deben cumplirse SIEMPRE.

## ⭐ 2. Principios fundamentales

- **El desbaste exige rigidez** — máquina débil = vibración.

- **El avance debe ser alto** — avance bajo = recorte y calor.

- **La evacuación de viruta es vital** — viruta atrapada = rotura.

- **La herramienta debe estar perfecta** — desgaste = riesgo.

## ⭐ 3. Tipos de fresas de desbaste cubiertas

- **Fresas de desbaste estándar**

- **Fresas de desbaste para aluminio**

- **Fresas de desbaste para acero**

- **Fresas de desbaste con rompevirutas**

## ⭐ 4. Requisitos previos

Antes de montar una fresa de desbaste:

- **Revisar desgaste**

- **Revisar geometría**

- **Revisar limpieza**

- **Revisar longitud expuesta**

- **Revisar balanceo**

## ⭐ 5. Seguridad en desbaste de aluminio

- **Prohibido usar sin evacuación de viruta**

- **Prohibido usar sin lubricación mínima**

- **Prohibido usar fresas con filo pulido dañado**

- Validación obligatoria:

  - filo

  - vibración

  - temperatura

## ⭐ 6. Seguridad en desbaste de acero

- **Prohibido usar sin refrigeración abundante**

- **Prohibido usar fresas con desgaste térmico**

- **Prohibido usar fresas sin rigidez suficiente**

- Validación obligatoria:

  - ruido

  - vibración

  - estabilidad

## ⭐ 7. Seguridad en operación

- **Prohibido usar RPM incorrectas**

- **Prohibido usar avances incorrectos**

- **Prohibido usar fresas desbalanceadas**

- **Prohibido usar fresas demasiado largas**

- Validación obligatoria:

  - ruido

  - vibración

  - temperatura

## ⭐ 8. Prohibiciones absolutas

- usar fresas sucias

- usar fresas desgastadas

- usar fresas con micro‑roturas

- usar fresas sin refrigeración

- usar fresas sin balanceo

- usar RPM incorrectas

- usar avances incorrectos

- ignorar vibración

- ignorar ruido

- ignorar temperatura

- operar sin validar

- operar sin registrar

## ⭐ 9. Objetivo final

Estas normas garantizan que las fresas de desbaste CNC‑CAT sean:

- seguras

- estables

- precisas

- profesionales

- industriales

- trazables

- confiables

Son las normas oficiales de seguridad de fresas de desbaste del ecosistema CNC‑CAT.
