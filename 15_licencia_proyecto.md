# 📘 **15\_licencia\_proyecto.md**

**Licencia oficial del proyecto CNC‑CAT / LINCAT**

# ⭐ 1. Propósito

Definir la **licencia legal** bajo la cual se distribuye el ecosistema CNC‑CAT, incluyendo:

- código fuente

- documentación

- diagramas

- interfaces

- especificaciones

- herramientas auxiliares

La licencia establece **qué puede hacer un usuario** y **qué no puede hacer** con el proyecto.

# ⭐ 2. Tipo de licencia

El proyecto CNC‑CAT utiliza una licencia **open‑source permisiva**, basada en:

### ✔️ **MIT License (modificada para uso industrial)**

Esto permite:

- uso comercial

- modificación

- distribución

- integración en sistemas propietarios

- uso en entornos industriales

Siempre que se mantenga:

- el aviso de copyright

- la licencia original

- la atribución al proyecto CNC‑CAT

# ⭐ 4. Extensiones específicas CNC‑CAT

Además de la licencia MIT estándar, CNC‑CAT añade:

### ✔️ 4.1 Prohibición de uso militar

El software **no puede** utilizarse en:

- sistemas de armamento

- control de drones militares

- automatización bélica

### ✔️ 4.2 Prohibición de uso en vigilancia masiva

No puede integrarse en sistemas destinados a:

- espionaje masivo

- vigilancia no autorizada

- control social automatizado

### ✔️ 4.3 Requerimiento de atribución visible

Toda UI o sistema derivado debe incluir:

Código

```
`Powered by CNC‑CAT`
```

# ⭐ 5. Derechos del usuario

El usuario puede:

- usar el software libremente

- modificarlo

- integrarlo en productos comerciales

- redistribuirlo

- crear forks

- traducirlo

- adaptarlo a hardware propio

# ⭐ 6. Obligaciones del usuario

El usuario debe:

- mantener la licencia

- mantener el copyright

- no eliminar la atribución

- no usarlo para fines prohibidos

- no reclamar propiedad exclusiva del proyecto original

- **sin responsabilidad legal**

El usuario asume todos los riesgos derivados del uso del software.
