# 📘 **73\_normas\_seguridad\_conos\_BT\_ISO\_HSK.md**

**Normas oficiales de seguridad para conos BT / ISO / HSK — CNC‑CAT / LINCAT**

## ⭐ 1. Propósito

Definir las **normas de seguridad para conos BT, ISO y HSK** utilizados en máquinas CNC‑CAT, garantizando:

- seguridad del operador

- integridad del husillo

- integridad del portaherramientas

- precisión geométrica

- estabilidad del mecanizado

- cumplimiento industrial

Los conos son el punto de contacto crítico entre el husillo y el portaherramientas. Si fallan, la máquina falla. Estas normas son **críticas** y deben cumplirse SIEMPRE.

## ⭐ 2. Principios fundamentales

- **El cono define la precisión** — cualquier imperfección se amplifica en el corte.

- **La limpieza es obligatoria** — suciedad = micras perdidas.

- **El contacto debe ser perfecto** — sin contacto total, hay vibración.

- **El desgaste es un riesgo** — desgaste = pérdida de rigidez.

## ⭐ 3. Tipos de conos cubiertos

- **BT**

- **ISO**

- **HSK**

## ⭐ 4. Requisitos previos

Antes de montar un cono:

- **Revisar limpieza absoluta**

- **Revisar desgaste**

- **Revisar deformación**

- **Revisar compatibilidad con el husillo**

- **Revisar torque o sistema de sujeción**

## ⭐ 5. Seguridad en conos BT

- **Prohibido usar conos sucios**

- **Prohibido usar conos con óxido**

- **Prohibido usar conos con desgaste en la brida**

- Validación obligatoria:

  - contacto

  - fijación

  - vibración

## ⭐ 6. Seguridad en conos ISO

- **Prohibido usar conos con deformación**

- **Prohibido usar conos sin torque adecuado**

- **Prohibido usar conos con suciedad en la brida**

- Validación obligatoria:

  - concentricidad

  - fijación

  - vibración

## ⭐ 7. Seguridad en conos HSK

- **Prohibido usar HSK sucios**

- **Prohibido usar HSK deformados**

- **Prohibido usar HSK sin contacto doble perfecto**

- Validación obligatoria:

  - rigidez

  - contacto

  - vibración

## ⭐ 8. Seguridad en operación

- **Prohibido usar RPM excesivas**

- **Prohibido usar herramientas largas sin reducción**

- **Prohibido usar herramientas desbalanceadas**

- Validación obligatoria:

  - ruido

  - vibración

  - temperatura

## ⭐ 9. Prohibiciones absolutas

- usar conos sucios

- usar conos deformados

- usar conos oxidados

- usar conos sin contacto perfecto

- usar herramientas fuera de tolerancia

- usar herramientas desbalanceadas

- ignorar vibración

- ignorar ruido

- ignorar temperatura

- operar sin validar

- operar sin registrar

## ⭐ 10. Objetivo final

Estas normas garantizan que los conos BT/ISO/HSK CNC‑CAT sean:

- seguros

- estables

- precisos

- profesionales

- industriales

- trazables

- confiables

Son las normas oficiales de seguridad de conos del ecosistema CNC‑CAT.
