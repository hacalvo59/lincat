# 📘 **67\_normas\_seguridad\_portaherramientas.md**

**Normas oficiales de seguridad para portaherramientas — CNC‑CAT / LINCAT**

## ⭐ 1. Propósito

Definir las **normas de seguridad para portaherramientas** utilizados en máquinas CNC‑CAT, garantizando:

- seguridad del operador

- integridad del husillo

- integridad de la herramienta

- estabilidad del mecanizado

- precisión geométrica

- cumplimiento industrial

El portaherramientas es el enlace crítico entre el husillo y la herramienta. Si falla, la máquina falla. Estas normas son **críticas** y deben cumplirse SIEMPRE.

## ⭐ 2. Principios fundamentales

- **El portaherramientas define la precisión** — cualquier error se amplifica en el corte.

- **La limpieza es obligatoria** — suciedad = desalineación.

- **El balanceo es seguridad** — vibración = destrucción.

- **La fijación es crítica** — una pinza floja es un accidente.

## ⭐ 3. Tipos de portaherramientas cubiertos

- **ER (pinza elástica)**

- **ISO/BT**

- **HSK**

- **Portapinzas hidráulicos**

- **Portapinzas térmicos**

- **Portaherramientas de precisión**

## ⭐ 4. Requisitos previos

Antes de montar un portaherramientas:

- **Revisar limpieza**

- **Revisar desgaste**

- **Revisar geometría**

- **Revisar torque de apriete**

- **Revisar compatibilidad herramienta‑portaherramientas**

## ⭐ 5. Seguridad en portaherramientas ER

- **Prohibido usar pinzas sucias**

- **Prohibido usar pinzas deformadas**

- **Prohibido usar tuercas sin torque adecuado**

- Validación obligatoria:

  - concentricidad

  - fijación

  - vibración

## ⭐ 6. Seguridad en portaherramientas ISO/BT

- **Prohibido usar conos sucios**

- **Prohibido usar conos con óxido**

- **Prohibido usar herramientas largas sin reducción de RPM**

- Validación obligatoria:

  - contacto cono‑husillo

  - fijación

  - vibración

## ⭐ 7. Seguridad en portaherramientas HSK

- **Prohibido usar HSK sin limpieza absoluta**

- **Prohibido usar HSK sin verificación de deformación**

- **Prohibido usar HSK sin lubricación mínima**

- Validación obligatoria:

  - contacto doble

  - rigidez

  - vibración

## ⭐ 8. Seguridad en portaherramientas hidráulicos

- **Prohibido usar sin presión adecuada**

- **Prohibido usar con fugas**

- **Prohibido usar sin herramienta compatible**

- Validación obligatoria:

  - presión

  - concentricidad

  - vibración

## ⭐ 9. Seguridad en portaherramientas térmicos

- **Prohibido usar sin calentamiento controlado**

- **Prohibido usar herramientas con óxido**

- **Prohibido usar sin tiempo de enfriamiento adecuado**

- Validación obligatoria:

  - concentricidad

  - fijación

  - vibración

## ⭐ 10. Seguridad en herramientas montadas

- **Prohibido usar herramientas desbalanceadas**

- **Prohibido usar herramientas demasiado largas**

- **Prohibido usar herramientas sin torque adecuado**

- Validación obligatoria:

  - longitud expuesta

  - rigidez

  - vibración

## ⭐ 11. Prohibiciones absolutas

- usar portaherramientas sucios

- usar portaherramientas deformados

- usar pinzas dañadas

- usar conos con óxido

- usar herramientas sin fijación

- usar herramientas sin balanceo

- ignorar vibración

- ignorar ruido

- ignorar temperatura

- operar sin validar

- operar sin registrar

## ⭐ 12. Objetivo final

Estas normas garantizan que los portaherramientas CNC‑CAT sean:

- seguros

- estables

- precisos

- profesionales

- industriales

- trazables

- confiables

Son las normas oficiales de seguridad de portaherramientas del ecosistema CNC‑CAT.
