# **6. Objetivo del índice**

Este índice permite:

- navegar el ecosistema completo,

- entender la estructura documental,

- localizar rápidamente cualquier módulo,

- mantener coherencia entre documentos,

- servir como base para documentación web (MkDocs, GitHub Pages, etc.).

Es el mapa maestro del proyecto.
